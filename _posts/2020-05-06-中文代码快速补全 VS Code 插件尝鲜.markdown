在现有编程语言中用中文命名标识符（变量、方法、类名等等）的一个常被鞭策的短板，是需要时常切换中英文。

趟路的多了，自然有能人开始把坑坑洼洼给填上。前不久刚发现的这个 VSC 插件就可以省去不老少中英切换的麻烦。

## 功能

一个源码文件中，往往有不少标识符重复出现。这个插件只需要在第一次输入时用外置的中文输入法，之后只需在英文状态下输入拼音，输入过的匹配拼音的中文字段就会出现在补全弹窗中。输入回车后，选中的提示项就被输入。

这个插件现在的实现机制很单纯，是对拼音进行匹配，并不基于语法信息，也就是不区分变量、方法、类，甚至注释。但由于单个源码文件中的中文字段总数有限，这一问题并不突出。

而且，相对于通用的外置中文输入法，由于它的可选项范围是当前源码文件中出现过的字段，拼音匹配的命中率也更高。
